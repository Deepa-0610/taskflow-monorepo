# @taskflow/shared

Shared types, utilities, and constants used across Taskflow packages.

## Exports

### Types
- `Task` - Task interface
- `User` - User interface
- `AuthUser` - Authenticated user interface

### Utilities
- `cn()` - Utility for combining Tailwind CSS classes

## Usage

In other packages:

```typescript
import { Task, User, cn } from '@taskflow/shared'
```
